🤖 CARTA-ERP AI Proxy

Cloudflare Pages Function koja proxy-a pozive iz CARTA-ERP browsera prema Google Gemini API-ju za ISO 9001 modul.

Endpoint

POST /api/iso/gemini

Request

{
  "feature": "audit_checklist",
  "context": { ... },
  "audit_id": "uuid (opcionalno)"
}

Status

Ako vidiš ovu stranicu — Cloudflare Pages je deployed ispravno. Test endpoint-a:

curl -X POST https://YOUR-PROJECT.pages.dev/api/iso/gemini \
  -H "Content-Type: application/json" \
  -d '{"feature":"audit_checklist","context":{"audit_year":2026,"scope_processes":["1.2 Vodjenje"],"nc_total":14,"capa_open":0,"documents_published":37}}'
